Mason Bickers 6 week hybrid plan 1.1

Hybrid Training + Half Marathon 6 Week Plan

Weekly Training Structure
Monday – [STRENGTH] Upper Strength
Tuesday – [RUN] Speed Run
Wednesday – [STRENGTH] Lower Strength
Thursday – [RUN] Easy Run
Friday – [STRENGTH] Upper Volume
Saturday – [RUN] Long Run
Sunday – [RUN] Tempo Run + [STRENGTH] Lower Hypertrophy

Running Pace Guide
Easy Pace: 5:15 – 5:45 /km
Long Run Pace: 4:45 – 5:00 /km
Tempo Pace: ~4:00 /km
10k Pace: ~3:52 /km
5k Pace: ~3:35 /km
Intervals: 3:20 – 3:30 /km

Strength Templates

Lower Strength Day
Back Squat – 5x5 RPE 8 180s rest
Romanian Deadlift – 3x6 RPE 8 150s rest
Walking Lunges – 3x10/leg RPE 7.5 90s rest
Hamstring Curl – 3x10 RPE 8 75s rest
Standing Calf Raises – 4x12 RPE 8 60s rest
Core – 3 sets RPE 7 60s rest

Lower Hypertrophy Day
Front Squat or Hack Squat – 4x8 RPE 8 120s rest
Hip Thrust – 4x8 RPE 8 120s rest
Single Leg RDL – 3x10/leg RPE 8 90s rest
Leg Extension – 3x12 RPE 8 75s rest
Hamstring Curl – 3x12 RPE 8 75s rest
Walking Lunges – 3x12 RPE 8 75s rest
Calf Raises – 4x12 RPE 8 60s rest
Core Carries – 3 rounds RPE 7.5 75s rest
Optional: Sled Push – 4x1 RPE 8.5 90s rest; Wall Balls – 3x20 RPE 8.5 75s rest

Upper Strength Day
Bench Press – 5x5 RPE 8 150s rest
Weighted Pull Ups – 4x6 RPE 8 120s rest
Overhead Press – 4x6 RPE 7.5 120s rest
Barbell Row – 4x8 RPE 7.5 105s rest
Biceps – 3x10 RPE 8 75s rest
Triceps – 3x10 RPE 8 75s rest

Upper Volume Day
Incline DB Press – 4x10 RPE 8 90s rest
Lat Pulldown – 4x10 RPE 8 90s rest
Lateral Raises – 4x15 RPE 8.5 60s rest
Cable Fly – 3x12 RPE 8 60s rest
Cable Curl – 3x12 RPE 8 60s rest
Triceps Pushdown – 3x12 RPE 8 60s rest

Week by Week Plan

Week 1
Monday: [STRENGTH] Upper Strength
Tuesday: [RUN] Speed Session (Week A) – 6 x 800m @ 3:30–3:35/km (90 sec rest), + 2km warm up + 2km cool down
Wednesday: [STRENGTH] Lower Strength
Thursday: [RUN] Easy Run 8 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume
Saturday: [RUN] Long Run 12 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 20 min @ ~4:00/km + [STRENGTH] Lower Hypertrophy
Target Weekly KM: ~34–38

Week 2
Monday: [STRENGTH] Upper Strength
Tuesday: [RUN] Speed Session (Week B) – 12 x 400m @ 3:20–3:25/km (60 sec rest), + 2km warm up + 2km cool down
Wednesday: [STRENGTH] Lower Strength
Thursday: [RUN] Easy Run 9 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume
Saturday: [RUN] Long Run 14 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 2 x 12 min @ ~4:00/km (3 min easy between) + [STRENGTH] Lower Hypertrophy
Target Weekly KM: ~38–42

Week 3
Monday: [STRENGTH] Upper Strength
Tuesday: [RUN] Speed Session (Week C) – 5 x 1km @ 3:40/km (2 min rest), + 2km warm up + 2km cool down
Wednesday: [STRENGTH] Lower Strength
Thursday: [RUN] Easy Run 10 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume
Saturday: [RUN] Long Run 15 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 25 min @ ~4:00/km + [STRENGTH] Lower Hypertrophy
Target Weekly KM: ~41–45

Week 4
Monday: [STRENGTH] Upper Strength
Tuesday: [RUN] Speed Session (Week D) – Ladder 400-600-800-1000-800-600-400 @ 3:30–3:40/km effort, jog recoveries 60–90 sec, + 2km warm up + 2km cool down
Wednesday: [STRENGTH] Lower Strength
Thursday: [RUN] Easy Run 10 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume
Saturday: [RUN] Long Run 17 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 3 x 10 min @ ~4:00/km (3 min easy between) + [STRENGTH] Lower Hypertrophy
Target Weekly KM: ~44–48

Week 5
Monday: [STRENGTH] Upper Strength
Tuesday: [RUN] Speed Session (Week B progression) – 14 x 400m @ 3:20–3:25/km (60 sec rest), + 2km warm up + 2km cool down
Wednesday: [STRENGTH] Lower Strength
Thursday: [RUN] Easy Run 10–11 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume
Saturday: [RUN] Long Run 18 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 30 min @ ~4:00/km + [STRENGTH] Lower Hypertrophy
Target Weekly KM: ~47–51

Week 6 (Deload + Absorb)
Monday: [STRENGTH] Upper Strength (reduce to ~80% volume)
Tuesday: [RUN] Speed Session (Week C reduced) – 4 x 1km @ 3:40/km (2 min rest), + 1.5km warm up + 1.5km cool down
Wednesday: [STRENGTH] Lower Strength (reduce to ~80% volume)
Thursday: [RUN] Easy Run 7–8 km @ 5:15–5:45/km
Friday: [STRENGTH] Upper Volume (reduce to ~80% volume)
Saturday: [RUN] Long Run 14 km @ 4:45–5:00/km
Sunday: [RUN] Tempo 20 min @ ~4:00/km + [STRENGTH] Lower Hypertrophy (reduce accessories)
Target Weekly KM: ~34–39

Execution Notes
- Keep Tuesday and Sunday quality sessions controlled.
- On heavy fatigue weeks, reduce accessory lifting volume before reducing run quality.
- Keep long run smooth and even, avoid surging in first half.
- If any signs of niggle or tendon tightness appear, swap one quality run for easy aerobic work.
